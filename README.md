# powerbi-sales-dashboard
## Overview
This project contains an interactive Power BI Sales Dashboard 
that analyzes and visualizes key sales metrics and business insights

##Dashboard preview
![Power BI Sales Dashboard](./screenshot.png)

## Features
- Total Sales & Revenue Analysis
- Sales by Region and Category
- Monthly/Yearly Sales Trends
- Top Performing Products
- Key Performance Indicators (KPIs)

## Tools Used
- Power BI Desktop
- Microsoft Excel (Data Source)

## File
- `sales.pbix` — Power BI report file

## How to Open
1. Download the `sales.pbix` file
2. Open with **Power BI Desktop**
3. Explore the interactive dashboard
